## Overview

This is where you create MCP server instances for specific teams. The MCP Servers tab provides access to 80+ pre-configured MCP servers from popular services like Vercel, Notion, ElevenLabs, and more, plus the ability to [add custom ones <Icon icon="up-right-from-square"/>](/admin-view/custom-mcp-servers).

<Warning>
  **Important**: By default, teams have NO access to any MCP servers. This follows a least-privilege security model by default.
</Warning>

## Available MCP Servers

Browse and configure from our extensive catalog of MCP servers:

- **Productivity**: Notion, Airtable, ClickUp, Cal.com, Calendly
- **Communication**: Discord, Agent Mail, ActiveCampaign
- **Development**: Vercel, GitHub, Atlassian, Context7
- **Data & Analytics**: Airtable, Baserow, Akkio
- **Media**: ElevenLabs, All-Images.ai
- **Finance**: CoinGecko, CoinMarketCap, DexScreener
- **Search & Research**: Brave Search, Exa, arXiv
- **And many more...**
