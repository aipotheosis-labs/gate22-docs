title: 'Configured MCP Servers'
description: 'View and manage all MCP servers that have been configured and are ready for use in your organization.'
icon: "gear"
---

**Configured MCP Servers** are instances you've set up for specific teams. These are MCP servers that have been successfully configured and are ready for team members to use.

Each configured MCP server instance is:
- **Team-specific** - Assigned to particular teams within your organization
- **Ready to use** - Fully configured with authentication and permissions
- **Accessible** - Available for team members to connect their accounts
- **Managed** - Monitored and maintained by administrators
